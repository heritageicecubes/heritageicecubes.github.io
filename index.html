HERITAGE ICE CUBES are a simple, cost-effective, and FUN way to pass on part of yourself to your offspring.

****Read on to learn how!****

(NOTE: Scroll down using your computer mouse or arrow buttons)
|
|
|
|
|
|

\/


First, empty a glass (or glasses) of clean spring water into your mouth. DO NOT SWALLOW
|
|
|
|
|
|
\/


Swish the water in your mouth to mix in your DNA (saliva)
|
|
|
|
|
|
|
|
|
|
|
|
|
|
|
\/
Do this for maybe 15 seconds
|
|
|
|
|
|
|
|
|
|
|
\/


After thoroughly infusing the water with your saliva, return it to the glass
|
|
|
|
|
|
|
|
|
|
|
\/


Empty the glass into a deep pan
|
|
|
|
|
|
|
|
|
|
|
\/


Place pan in freezer
|
|
|
|
|
|
|
|
|
|
|
|
|
|
|
|
|
|
\/
